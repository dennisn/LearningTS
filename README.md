# LearningTS

Project-based repository to learn TypeScript.

## Repository structure

- `projects/` contains all learning projects.
- Each subfolder in `projects/` is an independent TypeScript project.
- Project folder names follow this pattern:
  - `<Difficulty>_<TwoDigitNumber>_<ProjectName>`
  - Example: `Easy_01_TaskTrackerCli`

Difficulty prefixes:

- `Easy`
- `Med`
- `Hard`
